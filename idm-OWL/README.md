# idm-OWL

## Files in this subfolder
- [intavia_idm1.ttl](intavia_idm1.ttl) contains the complete IDM ontology expressed in OWL. It reuses components from CIDOC CRM, BioCRM and EDM
- [intavia_idm1.1.ttl](intavia_idm1.1.ttl) contains a subset of the ontology, only for internal use
- [intavia_idm_classes_1.ttl](intavia_idm1.1.ttl) contains a number of specific person relation roles as defined in the context of the InTaVia project that are useful for search and visualisation. These are defined as subclasses of bioc:Person_Relationship_Role
